Atividade 11

Usando Node.js, HTML e CSS, implemente uma API (Application Programming Interface) que receba o preço de 3 produtos na query string de uma URL e retorne:

  • O menor preço.
  • O maior preço.
  • A média aritmética dos preços informados.
Os preços devem ser informados num formulário HTML, usando o método GET do HTTP. Também teste a API usando o Postman/Insomnia.
